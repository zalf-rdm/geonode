{% trans "How to Cite the ZALF Repository" %}

{% trans "When using datasets, metadata, or other materials hosted in the ZALF Repository, we kindly request that you properly acknowledge the platform. Citation ensures transparency, credit to contributors, and supports the long-term sustainability of research data services." %}

{% trans "1. Cite the ZALF Repository Platform" %}

{% trans "If you want to reference the repository as an infrastructure or as a data source in general, please use the following recommended citation:" %}

ZALF Repository ({{ now|date:"Y" }}).
Leibniz Centre for Agricultural Landscape Research (ZALF).
Available at: https://repository.zalf.de

{% trans "2. Cite Individual Datasets" %}

{% trans "Each dataset published in the ZALF Repository includes a preferred citation format. To cite a specific dataset, use the citation provided on the dataset’s page, which typically includes:" %}

  • {% trans "Authors / data creators" %}
  • {% trans "Year of publication" %}
  • {% trans "Dataset title" %}
  • {% trans "ZALF Repository as the data publisher" %}
  • {% trans "Persistent Identifier (PID or DOI)" %}
  • {% trans "Version, if available" %}

{% trans "Example:" %}

Müller, A.; Schmidt, L.; Hoffmann, M. (2024).
Soil moisture measurements from the DEMM study area (v1.1).
ZALF Repository.
https://repository.zalf.de/dataset/abcd-1234

{% trans "3. Citing Datasets Without a Provided Format" %}

{% trans "If a dataset does not include a detailed citation suggestion, you may use the general pattern below:" %} 

Author(s) (Year).
Title of the dataset.
ZALF Repository.
Persistent Identifier (if available).
URL

{% trans "4. Referencing the Repository in Research Papers" %}

{% trans "When describing your data sources, methods, or Research Data Management practices, you may generally refer to the platform as follows:" %} 

"The research data used in this study were accessed through the ZALF Repository,
the institutional research data platform of the Leibniz Centre for Agricultural Landscape Research (ZALF)."

{% trans "5. Why Proper Citation Matters" %}

  • {% trans "Ensures that data creators receive appropriate credit." %}
  • {% trans "Improves scientific transparency and reproducibility." %}
  • {% trans "Helps track dataset usage and scientific impact." %}
  • {% trans "Supports long-term maintenance and infrastructure funding." %}
